# **App Name**: MSS Group BD

## Core Features:

- Homepage with Value Proposition: Visually clean sections including hero, trust indicators, about overview, services, product categories, buyer journey, capability snapshot, quality assurance, sustainability commitment, and final CTA.
- Product Catalog: Filterable catalog for Knitwear, Woven, Denim, and Sweater; Segments: Menswear, Womenswear, Kids & Newborn, Unisex; Indicate sourcing model.
- Company Profile Pages: About Us, Vision & Mission, Company Journey, Core Values with a professional corporate tone.
- Quality & Compliance Section: QA flow steps, inspection stages, worker safety & ethics, and compliance commitment.
- Contact & Inquiry Handling: Clean contact form with validation; submissions stored in Firestore.
- Inquiry Analysis Tool: Backend API route to provide the LLM with data to analyze submissions with internal, admin-only logic.
- Dark/Light Mode: Theme toggle respecting system preference and persisting user choice.

## Style Guidelines:

- Primary color: Deep blue (#224272) for trust and professionalism.
- Background color: Light gray (#F0F4F7) for a clean, modern feel.
- Accent color: Muted teal (#558B8B) for subtle highlights.
- Headline font: 'Space Grotesk' (sans-serif) for a modern look.
- Body font: 'Inter' (sans-serif) for readability.
- Use clean icons from Feather or a similar set.
- Subtle micro-animations for enhanced user experience.
